# Tuples

This folder contains Python programs focused on understanding and practicing tuples in Python through practical examples and foundational problem-solving exercises.

Tuples are immutable data structures used to store ordered collections of values that should not be modified after creation.

## Topics Covered

* Creating tuples
* Accessing tuple elements
* Tuple immutability
* Tuple length
* Iterating through tuples
* Tuple packing
* Tuple unpacking
* Single element tuples
* Tuple concatenation
* Tuple repetition
* Membership checking
* Tuple methods
* Tuple traversal
* Tuple searching and comparison logic
* Tuple slicing and reversing
* Tuple conversion techniques

## Programs

* `tuple_basics.py` → Demonstrates foundational tuple concepts such as creation, access, iteration, packing, unpacking, concatenation, repetition, and membership checking

* `tuple_methods.py` → Demonstrates commonly used tuple methods and tuple traversal examples

* `tuple_problems.py` → Contains tuple-based problem-solving exercises using loops, searching, counting, comparison, traversal, conversion, and slicing logic
