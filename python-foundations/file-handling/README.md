# File Handling

This folder contains Python programs focused on understanding and practicing file handling concepts through practical examples and beginner-level text-processing exercises.

File handling is an important backend development concept used for reading, writing, updating, and processing data from files.

The goal of this section is to strengthen file traversal, counting logic, text processing, and backend-oriented file-processing skills.

## Topics Covered

* Opening files
* Closing files
* Reading files
* File traversal
* Line-by-line processing
* Full content reading
* Text tokenization
* Relative file paths
* File problem solving

## Programs

* `file_basics.py` → Demonstrates foundational file handling concepts such as opening files, reading content, and closing files


## File Methods Practiced

* `open()`
* `close()`
* `read()`

## File Modes Practiced

* `'r'` → Read mode

## Problems Practiced

* Count number of lines in file
* Count number of words in file

## Key Concepts Learned

* File opening and closing
* File traversal
* Line-by-line processing
* Full content reading
* Text tokenization using `split()`
* Counting logic
* Relative file paths
* String processing from files
